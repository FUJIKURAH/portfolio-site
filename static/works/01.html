冷蔵庫管理サイト(ver 1.0)

- URL https://fridge-manager-portfolio.netlify.app/

概要

冷蔵庫の中身を管理するWebサイトです。
登録した食材の情報をもとに、一覧表示や管理ができるようになっています。

使用技術

JavaScript（Vanilla JS）
HTML / CSS
Vite

担当範囲

バックエンド（DB設計・API実装）は別エンジニアが担当し、フロントエンド全般（UI実装・API通信・画面設計）を担当しました。

API通信を行い、取得したデータをもとに画面表示を動的に切り替える処理や、 静的なUIの構築を行っています。

工夫した点・取り組み

MVCの考え方をベースに、機能ごとに責務を分けた構成を意識して設計しました。

API通信を使ったUI構築の経験が少なかったため、参考書やドキュメントを読みながら設計と実装を進めています。 将来的にフロントエンドフレームワークで作り直すことを前提に、まずは普段から使っているJavaScript（Vanilla JS）で処理の流れや設計の考え方を理解することを目的として開発しました。

また、バックエンドエンジニアとの連携にも力を入れました。 専門領域が異なるため、当初は仕様や意図の共有が難しい場面もありましたが、設計に必要な情報をスプレッドシートにまとめて可視化することで、認識のズレを減らし、円滑に連携できるよう工夫しました。

UI面では、冗長な説明を減らし、直感的に使い方が分かることを意識しています。 イラストやアイコン、配色を活用し、視覚的に理解しやすいデザインを心がけました。
